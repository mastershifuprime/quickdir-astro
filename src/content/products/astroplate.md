### Overview

Astroplate is the most popular free Astro starter template with over 3,000 GitHub stars. Built with Astro v6, Tailwind CSS v4, and TypeScript, it provides everything you need to launch a modern website.

- Built with Astro v6 and Tailwind CSS v4
- TypeScript support out of the box
- Blog with authors and categories
- SEO optimized with meta tags
- Dark mode support
- Full-text search with Pagefind
- Multilingual / i18n support
- Responsive design
- Open source and free to use
