### Overview

Storeplate is an Astro-powered Shopify Storefront boilerplate that helps you build blazing-fast e-commerce frontends with Tailwind CSS and Nanostores.

- Shopify Storefront API integration
- Tailwind CSS for styling
- Nanostores for lightweight state management
- Blazing-fast page loads with Astro
- E-commerce ready out of the box
- Open source and free to use
